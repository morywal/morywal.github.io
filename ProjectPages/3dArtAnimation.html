Miles Morales 3D Animation and Model

My Role: [3D Artist, Rig, and Animator]
Date Created: [May 2024]
Context: [Created for Cinema 4D, California Lutheran University]
Tech Stack: [Cinema 4D]

The Project

The initial assignment was to 3D model a single static character. Inspired by Across the Spider-Verse, I decided to push this project far beyond the requirements, setting a personal goal to not only model Miles Morales but to use that model as the centerpiece for a complete, one-minute animated short.

The Process & Goals

My goal was to manage a full animation pipeline, which presented two major technical hurdles.

1. New Shading System: Mid-project, Cinema 4D released a brand new Toon Shading system. With no class time dedicated to it, I had to teach myself the new tool, working over Thanksgiving break to experiment with texture mapping and node graphs to achieve the desired comic-book effect.

2. The Render Farm: A one-minute, stylized animation creates a massive amount of render data. To complete the project by the deadline, I built an ad-hoc render farm by networking 20 of the lab's computers together. I managed this 10-hour, 20-machine render process to deliver the final video. This project was a crash course in technical resourcefulness and problem-solving.
